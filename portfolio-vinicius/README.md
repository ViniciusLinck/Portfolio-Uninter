# Portfolio Vinicius

## Objetivo

Este projeto é um portfólio pessoal e currículo online criado para uma atividade prática de Fundamentos da Programação Web. O site apresenta informações sobre Vinicius, sua formação, conhecimentos técnicos, projetos e um formulário de contato com validação em JavaScript puro.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript puro

O projeto não utiliza frameworks CSS, bibliotecas JavaScript ou ferramentas externas.

## Estrutura de pastas

```text
portfolio-vinicius/
  index.html
  css/
    style.css
  js/
    script.js
  README.md
```

## Como executar localmente

1. Abra a pasta `portfolio-vinicius`.
2. Dê dois cliques no arquivo `index.html`.
3. O site será aberto no navegador.

Também é possível usar a extensão Live Server no VS Code, caso esteja disponível.

## Observações

- O menu usa navegação por âncoras para as seções Sobre mim, Formação, Portfólio e Contato.
- O formulário valida nome, e-mail e mensagem antes de simular o envio.
- O site possui botão para alternar entre tema claro e tema escuro.
- O layout é responsivo e funciona em desktop, tablet e celular.
