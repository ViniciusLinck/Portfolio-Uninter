# Portfólio Vinicius Dias Linck

## Objetivo

Este projeto foi desenvolvido como trabalho acadêmico da disciplina de Aplicação Web e Móveis. A proposta é criar um portfólio pessoal utilizando apenas HTML, CSS e JavaScript simples.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Estrutura de pastas

```text
portfolio-vinicius/
  index.html
  css/
    style.css
  js/
    script.js
  README.md
```

## Funcionalidades

- Navegação por âncoras entre as seções do currículo.
- Alternância entre temas claro e escuro, com preferência salva no navegador.
- Menu responsivo para dispositivos móveis.
- Links diretos para e-mail, telefone e LinkedIn.
